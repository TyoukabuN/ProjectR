### 存档数据
- 通关卡厄思后可保留角色的「存档数据」（即该角色的卡组构筑成果）
- 存档可带入副本刷材料模式
- 理想存档目标：复制关键卡牌 + 触发灵光/神光 + 移除低效卡牌 + 获得更优装备

### 全局 Buff
- 每局开始前三选一选取全局 buff（类杀戮尖塔遗物）
- 困难模式附带全局 debuff

### 地图事件
- 篝火：训练（三选一获取中立牌）、商店（购买/消除诅咒牌）
- 特殊事件：可能提供卡牌、复制、装备奖励

## 九、对本游戏策划的借鉴点

### 值得直接参考的

| 机制 | 卡厄思做法 | 对本游戏的启示 |
|------|-----------|--------------|
| **快速** | 打出不消耗敌方行动计数器 | 适配我方"魔力值不减混乱度"的设计，可引入「快速」：打出不增加混乱度 |
| **终极** | 打出后立即结束出牌阶段 | 已采用，可作为高爆发/高风险卡的设计锚点 |
| **蒸发** | 不用就销毁，自动触发 | 已采用，适合"限时触发法式"的表达 |
| **储存** | 保留费用到下回合叠加 | 已采用，适合"蓄力型法式"的节奏设计 |
| **天上** | 特定条件自动打出 | 可用于"高阶吟唱的自动触发"，如"使用3费以上法式时，自动附加改良" |
| **连击** | 使用队友卡时触发 | 可参考设计"多道施法触发连锁"机制 |
| **击破+AP奖励** | 打破韧性后奖励费用 | 可对应"完全化解一张敌方牌后，返还部分魔力值" |
| **X费机制** | 消耗全部费用倍增效果 | 适合本游戏"全力一击"型法式设计 |

### 谨慎参考（机制差异较大）

| 机制 | 说明 |
|------|------|
| **灵光一闪进化** | 本游戏卡牌不走成长路线，不建议照搬 |
| **韧性槽系统** | 本游戏敌方是行为牌机制，不直接对应韧性概念 |
| **压力/崩溃** | 本游戏已有「混乱度」对应，无需重复引入 |

### 开荒攻略的核心设计观点
> 来源：gamekee wiki 开荒攻略 https://www.gamekee.com/czn/680816.html

| 维度 | 卡厄思做法 | 对本游戏的设计启示 |
|------|-----------|------------------|
| **省费/断费能力** | 前期费用有限，零费贡献角色（如维若妮卡稳定0费150%）和回费角色（米卡）优先级最高 | 法师初期法力值稀缺，应设计"零消耗触发"的低费法式组合，以及"完成特定组合返还费用"的激励机制 |
| **强化卡连动** | "其他角色使用卡牌时发动效果"的强化卡免费做贡献 | 可设计"吟唱中途自动追加效果"的被动语言卡，打出其他卡牌时自动连锁 |
| **过滤卡组** | 毕业后过牌（快速循环卡组）是最保值的能力 | 抽牌/过滤类法式在后期越来越强，应在设计时为其预留足够强度空间 |
| **角色定位** | 主C吃费出伤，副C不吃费做贡献，奶妈负责压力管理 | 战斗角色应有明确资源定位：高费爆发型/零费持续型/回血稳定型 |

## 十、装备词条系统（第二赛季新增）

> 来源：gamekee wiki https://www.gamekee.com/czn/692900.html

### 基本规则
- 装备可附加 **1-2 条随机词条**以强化效果
- 词条**绑定槽位而非装备本体**：替换装备后词条全部保留
- 每条词条占用 **10 PT**（存档点数上限），最多同时占用 40 PT
- 词条必定保留在存档中，需注意 PT 上限管理

### 词条获取方式
| 途径 | 说明 |
|------|------|
| 事件随机 | 部分事件奖励直接带词条的装备 |
| 竞技场重铸 | 打完2场战斗后可对任意装备重铸1条词条（覆盖原有词条） |
| 神之锤词条 | 通关核心层后获得，与重铸词条并列不覆盖 |

### 代表性词条示例（可参考设计装备/建筑加成效果）

| 词条效果 | 设计启示 |
|----------|----------|
| 卡牌「保留」「消灭」时对随机敌人造成100%固定伤害（每回合2次） | 关键字本身可作为伤害触发器 |
| 发动费用3以上的卡牌时，获得AP1（每回合1次） | 高费触发奖励：出强力牌有额外费用回馈 |
| 战斗开始时储存1 | 开局词条：战斗开局预置资源 |
| 在持有AP1以上状态下回合结束时，使所有敌人脆弱1 | 持有资源型：保留费用触发全场Debuff |
| 若持续1回合未以卡牌对敌人造成伤害，下回合伤害量+50% | 蓄力型：跳过进攻回合后爆发 |
| 战斗开始时若持有卡牌≥33张，攻击卡牌伤害+30% | 卡组规模判断：大卡组/小卡组走不同收益路线 |
| 回合开始和结束时若仍持有护盾，对所有敌人造成固定伤害100% | 护盾残留奖励：防御过量时转化输出 |
| 使用能力抽取自身攻击卡牌时，该卡牌持续1回合伤害+50% | 检索+增幅联动：主动检索牌有伤害加成 |

### 神之锤词条
- 效果从**本局未出现的装备效果**中随机抽取，范围极广
- 稀有装备词条为蓝色，传说为黄色，神话为紫色
- 高方差、高随机性，是局内最强随机增益来源之一

### 对本游戏的借鉴点
- **建筑/塔楼附加词条**：塔楼可随机附加类似效果（如"每回合开始时若有护盾残留，化解1层混乱度"）
- **法坛强化词条**：法坛可附加触发型增益（如"完成3费以上法式时，额外返还1点魔力值"）
- **高费触发奖励机制**：强力法式可附带"触发后返还魔力值/抽牌/降低混乱度"的额外效果，鼓励玩家追求强力组合而非频繁出小牌

## 十一、机制深度解析（进阶攻略）

> 来源：gamekee wiki 进阶攻略

### 1. 增加 vs +（乘算 vs 加算）

| 写法 | 计算方式 | 适合场景 |
|------|----------|----------|
| **增加XX%** | 乘算：基础伤害 × (1+XX%) | 少次高倍率的卡牌，收益固定 |
| **+XX%** | 加算：卡牌倍率直接 +XX% | 多次低倍率的卡牌，对低初始倍率收益大 |
| **不屈/削弱** | 减伤：-XX% 为加算，减少XX% 为乘算 | -伤害量对抗多段低倍率更有效；减少伤害量对抗少段高倍率更好 |

**对本游戏启示**：设计法式增伤时需明确区分乘算词条和加算词条，配合不同类型的法式会有截然不同的收益，这是卡组深度的来源之一。

### 2. 抽取 vs 移动

| 机制 | 规则 | 限制 |
|------|------|------|
| **抽取** | 从抽牌堆抽指定卡牌；无则触发洗牌再抽；本回合已用的牌不参与洗牌 | 每张卡每回合只能抽一次，有上限 |
| **移动** | 从抽牌堆或坟墓直接加入手牌，无视本回合是否已使用 | 无次数限制，上限更高 |

- 被效果**丢弃**进坟墓的牌（非使用），可通过抽取触发洗牌后再拿回来
- 从抽牌堆移动的牌，能触发"能力抽取"类的装备效果

**对本游戏启示**：语言卡的"检索"类效果需区分从牌库移入手牌（移动）和触发洗牌后抽取，两者在一回合连续触发时有本质差异。

### 3. 战斗中压缩牌库的手段

| 方式 | 说明 | 特点 |
|------|------|------|
| **消灭** | 使用后移出游戏，不进坟墓/抽牌堆 | 高效压缩，0费消灭卡最佳 |
| **强化** | 使用后本场全程生效，消失于牌库 | 需费用，迟到则是废牌 |
| **保留** | 留在手牌=不进循环，等效压缩 | 操作压缩，适合无抽牌能力的角色 |

# 三、核心玩法设计

---

## 3.1 职业系统

玩家选择的**魔法师职业**本质上决定了其起始卡组的构成与魔力值的产出方式（类似杀戮尖塔中不同角色持有完全不同的卡池）。不同职业在卡牌风格、施法侧重和魔力来源上各具特色：

| 职业示例 | 擅长方向 | 魔力产出特色 |
|----------|----------|-------------|
| **精灵吟唱师** | 精准单体、持续灼烧 | 每回合自动生成1张「精灵之息」魔力卡 |
| **龙语术士** | 大范围地形改造 | 召唤龙魂造物，造物每回合产出「龙能」魔力卡 |
| **魔神契约者** | 强力增幅、高风险附魔 | 打出特定魔法卡触发「契约反馈」，瞬间生成魔力卡 |

> 职业的核心差异不仅是卡池，更是**魔力值的获取方式**——不同职业需要用截然不同的策略来维持施法节奏。

---

## 3.2 卡牌施法系统

魔法吟唱使通过卡牌的组合来完成法术的吟唱与施展。卡牌分为以下类型：

| 卡牌类型 | 说明 |
|----------|------|
| **语言卡** | 描述施法所用的魔法语种，决定魔法的属性流派与基础规则（如精灵语、魔神语、龙语） |
| **姿势卡** | 描述施法者的身体动作，决定魔法的释放形态（如扩散、集中、持续、爆发） |
| **材料卡** | 描述魔法的物质构成（如火焰、寒冰、石土、风暴、圣光、暗影） |
| **魔力卡** | 职业专属的费用来源卡牌，打出即转化为魔力值，部分带有额外触发效果 |
| **改良卡** | 临时插入当前法式中，增幅或改变魔法效果，存在一定失败风险 |

### 吟唱阶段——组合出牌生成魔法卡

吟唱阶段即出牌阶段：玩家从手牌中按照法式公式打出对应的组合卡牌，即时生成一张**魔法卡**，并在本回合立即使用。

同一魔法效果可由**多种不同卡牌组合**来生成，不同组合方式会给生成的魔法卡附加 Buff：

| 目标魔法 | 标准组合 | 替代组合 | 替代组合的 Buff |
|----------|----------|----------|----------------|
| 火焰弹 | 精灵语+挥臂+烈焰 | 龙语+挥臂+烈焰 | 伤害+50%，但消耗+2 |
| 地形隆起 | 龙语+绕转+石土 | 精灵语+绕转+石土 | 范围缩小，但冷却减半 |

### 施法公式示例

- `语言：精灵语` + `姿势：挥臂` + `材料：烈焰` = 精灵火焰弹（精准单体、灼烧持续）
- `语言：龙语` + `姿势：绕转` + `材料：石土` = 龙脊隆起（地形更宽广、持续时间更长）
- `语言：魔神语` + `姿势：压掌` + `材料：寒冰` = 冰封诅咒（附魔效果更强、但对友方也有冰冻风险）

公式可在战前整理，战斗中从牌库抽卡，凑出需要的组合以应对战局。

### 临时改良

在标准法式中临时插入改良卡，可增强效果（如范围扩大、伤害翻倍），但：
- 改良卡有一定失败概率，失败时魔法可能走形或产生反效果
- 改良卡消耗后不回收，具有一次性风险成本

### 卡牌关键字

参考卡厄思梦境等卡牌游戏，本游戏的卡牌使用以下标准关键字标注特殊规则：

| 关键字 | 效果说明 |
|--------|----------|
| **保留** | 回合结束时不进入弃牌堆，保留在手牌中 |
| **消灭** | 使用后直接移出游戏，不进入弃牌堆（一次性强力卡） |
| **开战** | 战斗开始时自动置于抽牌堆顶端，必先抽到 |
| **回收** | 进入弃牌堆时自动回到手牌（触发一次后移除该词条） |
| **终极** | 使用后立即结束本回合吟唱阶段 |
| **蒸发** | 回合结束时自动打出并消耗，无需手动使用 |
| **储存** | 保留本卡消耗的魔力值至下一回合叠加使用 |

## 3.3 单位与目标系统

战场场地上存在可交互的**单位**，魔法卡或其他行为牌均可选择目标作用对象（类似炉石传说中选择随从、杀戮尖塔中选择敌人）。

**友方单位**

| 单位类型 | 说明 |
|----------|------|
| **生物单位** | 玩家召唤的魔法造物、守卫精灵等，可驻守阵地或执行特定行为 |
| **建筑单位** | 城墙、塔楼、魔法炮台等，有耐久度，受损会影响玩家能力 |

**敌方单位**

| 单位类型 | 说明 |
|----------|------|
| **生物单位** | 各类敌方士兵、怪物，在场地格子上移动并执行行为牌 |
| **建筑单位** | 攻城器、弩车阵地等，固定位置持续输出或施加特殊效果 |

## 3.4 场地结构——六边形格子

战场以**六边形格子**为基础单位构成。选择六边形的原因：

- 每格有 **6 个相邻方向**，移动与范围判定更自然流畅，无斜线距离歧义
- 适合表现包围、迂回、突破等丰富的战场态势
- 城墙沿六边形边缘排布，攻城路线选择更多样

单位在格子间移动，地形魔法可改变指定格子的属性（沼泽、冰封、燃烧等）。

## 3.5 关卡即对手——双方对牌机制

关卡中的敌人不仅仅是数值上的障碍，其**每一个行为都对应一张行为牌**，公开亮出后在下一回合执行。

| 敌方牌示例 | 效果 |
|------------|------|
| **冲锋** | 某小队本回合移动速度+2，强行突破前线地形障碍 |
| **箭雨覆盖** | 对城墙某段造成持续伤害，压制城防反击 |
| **包围推进** | 多路同时施压，分散玩家吟唱焦点 |
| **攻城器进场** | 召唤投石车，对城墙本体造成结构伤害 |
| **溃退重整** | 敌方撤退，为下一波更强的推进蓄力（需玩家判断是否追击还是补强防御） |

玩家可以通过侦察、预判牌面，提前构造对应的法式应对。

**敌方行为牌的数量由"混乱度"决定**（详见 [gd-04-系统设计.md](gd-04-系统设计.md) 4.7 混乱度系统）。

## 3.6 城防结构

城防由多个可交互节点构成：

- **城墙本体**：有耐久度，可通过强化魔法提升，被攻城器打击时受损
- **塔楼**：玩家吟唱施法的立足点，塔楼受损会影响吟唱效率（手牌上限/抽牌速度）
- **城门**：核心保护目标，城门被突破即战败
- **护城区域**：城墙外的缓冲地带，可在此设置地形陷阱

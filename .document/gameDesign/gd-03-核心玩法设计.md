## 3.2 卡牌施法系统

【魔法吟唱使】通过打出卡牌来完成法术的施展。卡牌分为以下类型：

| 卡牌类型 | 说明 |
|----------|------|
| **【魔法卡】** | 施法核心卡，卡组主体。打出即施放对应魔法，有基础效果，无需额外组合即可独立使用 |
| **【语言卡】** | 本回合变体触发卡。打出后，本回合内所有【魔法卡】变为对应语言的变体版本（仅限本回合生效） |
| **【姿势卡】** | 可选词条附加卡。搭配【魔法卡】使用，给本次施法追加释放形态相关的词条效果 |
| **【材料卡】** | 可选词条附加卡。搭配【魔法卡】使用，给本次施法追加元素属性相关的词条效果 |
| **【改良卡】** | 给本次施法增幅或改变效果，存在一定失败风险 |

### 魔法卡——施法核心

【魔法卡】是玩家【卡组】的核心构成。卡组中存放的均为**原始版本**，打出即可施放基础效果。

**一次“朴素施法”**（仅打出魔法卡）：
> 打出【火焰弹】= 基础伤害效果，立即生效

**一次“强化施法”**（配合词条卡）：
> 先打出【龙语（语言卡）】→ 再打出【绕转（姿势卡）】→ 打出【火焰弹（魔法卡）】
> = 火焰弹变为「龙焰弹」变体，并附加「绕转」词条效果

### 语言卡——本回合变体

【语言卡】打出后，在本【回合】内改变后续打出的【魔法卡】：

- 改变的是**本回合的施法形态**，回合结束后失效
- 卡组中的【魔法卡】本身不发生变化，下一回合仍以原始版本抽出
- 不同语言变体赋予相同魔法不同的属性或特性加成（如精灵语偏精准、龙语偏范围、魔神语偏附魔）

### 词条卡——可选增强

【姿势卡】与【材料卡】统称为**词条卡**，是可选的施法增强手段：

- 不需要词条卡也能正常施法
- 词条卡赋予本次施法额外效果（如：范围扩大、持续伤害、连锁触发等）
- 多张词条卡可叠加使用，但每张均需消耗对应【魔力值】

### 弃牌换卡——筛选手牌

每个【回合】玩家拥有固定次数的**【弃牌机会】**，可以在【吟唱阶段】前主动弃置手中不需要的卡牌，并从【牌库】补抽同等数量的新牌。

- 默认【弃牌机会】次数待定（参考值：2次/回合）
- 每次可弃置任意数量的手牌
- 【弃牌机会】用完后无法再换牌，需谨慎使用
- 弃置的卡牌进入【弃牌堆】

> **设计意图**：将“能否凑齐想要的施法组合”从**纯运气**转变为**资源决策**——玩家用【弃牌机会】换取更优的手牌组合，产生策略深度。

### 改良卡——风险增幅

在施法中临时插入【改良卡】，可增强效果（如范围扩大、伤害翻倍），但：
- 【改良卡】有一定失败概率，失败时魔法可能走形或产生反效果
- 【改良卡】消耗后不回收，具有一次性风险成本

### 卡牌关键字

参考卡厄思梦境等卡牌游戏，本游戏的卡牌使用以下标准【卡牌词条】标注特殊规则：

| 【卡牌词条】 | 效果说明 |
|--------|----------|
| **【保留】** | 【回合】结束时不进入【弃牌堆】，保留在【手牌】中 |
| **【消灭】** | 使用后直接移出游戏，不进入【弃牌堆】（一次性强力卡） |
| **【开战】** | 战斗开始时自动置于【抽牌堆】顶端，必先抽到 |
| **【回收】** | 进入【弃牌堆】时自动回到【手牌】（触发一次后移除该【卡牌词条】） |
| **【终极】** | 使用后立即结束本【回合】【吟唱阶段】 |
| **【蒸发】** | 【回合】结束时自动打出并消耗，无需手动使用 |
| **【储存】** | 保留本卡消耗的【魔力值】至下一【回合】叠加使用 |

## 3.3 单位与目标系统

战场场地上存在可交互的**【单位】**，【魔法卡】或其他【行为牌】均可选择目标作用对象（类似炉石传说中选择随从、杀戮尖塔中选择敌人）。

**友方【单位】**

| 单位类型 | 说明 |
|----------|------|
| **【生物单位】** | 玩家召唤的魔法造物、守卫精灵等可馻守阵地或执行特定行为 |
| **【建筑单位】** | 【城墙】、【塔楼】、魔法炮台等，有耐久度，受损会影响玩家能力 |

**敌方【单位】**

| 单位类型 | 说明 |
|----------|------|
| **【生物单位】** | 各类敌方士兵、怪物，在场地【格子】上移动并执行【行为牌】 |
| **【建筑单位】** | 攻城器、弩车阵地等，固定位置持续输出或施加特殊效果 |

## 3.4 场地结构——六边形格子

战场以**【六边形格子】**为基础单位构成。选择六边形的原因：

- 每格有 **6 个相邻方向**，移动与范围判定更自然流畅，无斜线距离歧义
- 适合表现包围、迂回、突破等丰富的战场态势
- 【城墙】沿六边形边缘排布，攻城路线选择更多样

【单位】在【格子】间移动，地形魔法可改变指定【格子】的属性（沼泽、冰封、燃烧等）。

## 3.5 关卡即对手——双方对牌机制

关卡中的敌人不仅仅是数值上的障碍，其**每一个行为都对应一张【行为牌】**，公开亮出后在下一【回合】执行。

| 敌方牌示例 | 效果 |
|------------|------|
| **【冲锋】** | 某小队本【回合】移动速度+2，强行突破前线地形障碍 |
| **【箭雨覆盖】** | 对【城墙】某段造成持续伤害，压制城防反击 |
| **【包围推进】** | 多路同时施压，分散玩家吟唱焦点 |
| **【攻城器进场】** | 召唤投石车，对【城墙】本体造成结构伤害 |
| **【溃退重整】** | 敌方撤退，为下一波更强的推进蓄力（需玩家判断是否追击还是补强防御） |

玩家可以通过侵察、预判牌面，提前构造对应的【法式】应对。

**敌方【行为牌】的数量由【混乱度】决定**（详见 [gd-04-系统设计.md](gd-04-系统设计.md) 4.7 【混乱度】系统）。

## 3.6 城防结构

城防由多个可交互节点构成：

- **【城墙】本体**：有耐久度，可通过强化魔法提升，被攻城器打击时受损
- **【塔楼】**：玩家吟唱施法的立足点，【塔楼】受损会影响吟唱效率（【手牌】上限/抽牌速度）
- **【城门】**：核心保护目标，【城门】被突破即战败
- **【护城区域】**：【城墙】外的缓冲地带，可在此设置地形陷阱

### 设计参考要点

**被动触发条件类型归纳**（对我方机制设计的参考）：

| 触发类型 | 代表伙伴 | 设计意义 |
|---|---|---|
| 生成牌时 | Bria、Anteia、Carroty | 将"创造临时牌"行为与伙伴收益绑定，鼓励生成型角色 |
| 暴击时叠层 | Serithea | 暴击率投资有双重回报（伤害+伙伴层数） |
| 弃牌时 | Kiara | 将弃牌（通常是损失）转化为伙伴增益 |
| 使用Skill/Upgrade牌 | Erica、Lilian | 引导玩家在攻击牌之外重视辅助类型牌 |
| Retain牌数量 | Noel | Retain持有数量本身成为伤害系数 |
| 摸牌时（每回合首次） | Nyx、Solia | 限制为首次，避免无限触发，保持平衡 |
| Ravage触发 | Peko | 与Sereniel的Homing Laser完美联动，形成"角色+伙伴"专属搭配 |
| 连续使用自身牌 | Scarlet | 鼓励单角色集中出牌而非轮流分散 |

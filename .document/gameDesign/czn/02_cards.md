## 四、卡牌类型

| 类型 | 说明 |
|------|------|
| **攻击卡牌** | 直接对敌造成伤害 |
| **技能卡牌** | 一次性生效（如加盾、抽牌） |
| **强化卡牌** | 打出后本场战斗永续生效，使用后从牌库消失 |
| **状态异常牌** | 敌方施加给玩家的负面牌（在牌库/手牌/坟墓中带 debuff） |
| **诅咒卡牌** | 占据手牌位或持续施加负面效果，可在商店消除 |

## 五、卡牌关键字（机制词条）完整列表

### 卡牌行为关键字

| 关键字 | 原游戏效果说明 |
|--------|--------------|
| **保留** | 回合结束时此卡牌不会进入坟墓，留在手牌中 |
| **蒸发** | 回合结束时，若在手牌中则自动消灭（不用就销毁） |
| **消灭** | 使用后移入「消灭牌堆」，不经过坟墓，永久移出循环 |
| **消灭2** | 使用2次后移入消灭牌堆 |
| **回收** | 进入坟墓时回到手牌；触发一次后移除该词条 |
| **回收3/4** | 同上，可触发3次/4次 |
| **开战** | 战斗开始时自动置于抽牌堆顶端（必先抽到） |
| **终极** | 使用后立即结束本回合 |
| **快速** | 使用该卡牌不会减少敌方行动计数器 |
| **天上** | 使用其他2费以上的卡牌时，自动打出本卡 |
| **连击** | 使用其他同伴卡牌时，发动本卡牌效果 |
| **储存** | 保留本卡消耗的 AP 至下回合叠加（层数减少1） |
| **主导** | 回合开始时50%概率费用-1；使用其他卡牌时移除效果 |
| **唯一** | 仅能持有1张同属性的唯一卡牌（无法复制） |
| **灵感** | 通过能力移入手牌时触发灵感效果（非普通抽取不触发） |
| **感应** | 仅在被抽到时发动效果 |
| **击杀** | 击杀目标时发动效果 |
| **灵感** | 通过能力移入手牌时触发（回合开始普通抽牌不触发） |

### 战斗属性关键字

| 关键字 | 原游戏效果说明 |
|--------|--------------|
| **弱点攻击** | 使用与敌人同属性卡牌攻击，伤害提升+韧性额外减少 |
| **韧性伤害** | 对敌方韧性槽造成的伤害量 |
| **击破** | 敌方韧性归零时触发，获得1点AP |
| **粉碎** | 攻击持有护盾目标时，伤害+20% |
| **标记** | 使用攻击卡牌时追加额外100%攻击，造成0韧性伤害，标记-1 |
| **连续** | 上一张为同属性卡牌时触发额外效果 |
| **金属化** | 使用卡牌获得护盾时，对随机敌人造成基于防御80%伤害 |
| **余烬** | 攻击处于击破状态的目标时，按余烬层数使伤害+30%；发动后移除（最多5层） |
| **破坏** | 击破目标或目标处于击破状态时发动效果 |
| **分贝** | 洗牌时，所有友军对随机敌人造成额外攻击50%；按层数伤害+50%（最多5层），发动后移除 |
| **开始狩猎** | 使用锁定攻击卡牌时，对目标额外攻击50%；按层数额外攻击伤害+30%（最多5层） |
| **凝聚** | 使用百丽儿攻击卡牌时，按层数造成额外伤害50%；发动后移除（最多3层） |
| **上膛** | 投射机伤害+50%；回合结束移除（最多3层） |
| **HP减少** | 按HP上限比例减少HP，无视护盾 |
| **伤害减少** | 所受伤害-15%，触发后-1 |

### X费卡牌
- **X费**：消耗当前全部 AP，按实际消耗量倍率发动效果，上限越高效果越强

### 角色专属机制词条（示例）
| 词条 | 所属角色 | 说明 |
|------|----------|------|
| **黑云姿态** / **黑云剑法** | 琳 | 处于黑云姿态时触发剑法额外效果，使用其他角色攻击卡牌时姿态移除 |
| **任务** / **任务卡牌** | 凯西乌斯 | 战斗中满足条件时生成任务卡牌（三连击/同花顺/葫芦），完成任务获得奖励 |
| **创造物** | 奥尔莱亚 | 召唤三种创造物卡牌；手牌中持有相同创造物★时可进化合并，最多3阶段 |
| **手枪弹药** / **投射机** / **发射器弹药** | 麦格农等 | 角色专属弹药类型，特定卡牌联动触发额外效果 |
| **银色之幕** | — | 银色之幕叠至3层时，对所有敌人造成防御力400%固定伤害，发动后全部移除 |

## 六、状态词条

### 正面状态（Buff）

| 词条 | 效果 |
|------|------|
| **护盾** | 本回合结束前，受到的伤害减去护盾量（护盾完全挡住则不增压力） |
| **固定护盾** | 无视持续效果的护盾 |
| **护盾保留** | 护盾持续到下一回合 |
| **不屈** | 所受伤害-20%，回合结束失去（可叠加） |
| **决心** | 护盾获取量+20%，回合结束失去（可叠加） |
| **士气** | 伤害量+20%（可叠加，正面） |
| **愤怒** | 伤害量+20%（可叠加） |
| **反击** | 受到伤害时，对攻击者造成防御100%的固定伤害；完美格挡时150% |
| **夹击/合击** | 友军攻击时，以攻击力100%一同攻击 |
| **闪避** | 避免1次攻击伤害，触发后-1 |
| **免疫** | 不会获得减益效果，触发后-1 |
| **治愈** | 按防御力比例恢复HP，根据压力量减少随机角色压力1-6 |
| **结晶化** | 回合结束时，获得相当于层数×固定护盾20% |

### 负面状态（Debuff）

| 词条 | 效果 |
|------|------|
| **脆弱** | 受到伤害+50%，触发后-1 |
| **士气减少** | 伤害量-20%，回合结束失去所有层数（可叠加） |
| **削弱** | 伤害量-25%，触发后-1 |
| **伤口/损伤** | 护盾获取量-50%，触发后-1 |
| **痛苦** | 回合结束时，固定持续伤害50%×层数，触发后-1 |
| **痛苦刻印** | 获得痛苦时，追加痛苦2层 |
| **惩戒** | 获得护盾时，受到攻击力200%的固定伤害，-1 |

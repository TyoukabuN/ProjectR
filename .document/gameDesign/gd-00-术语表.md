## 职业

| 术语 | 定义 |
|------|------|
| **【城之魔法使】** | 以城防为核心场景，擅长城墙强化、城防武器、地形控场，核心机制为【建筑】系统。 |
| **【战术魔法吟唱使】** | Magic caster 以大规模野战为核心场景，擅长广域攻击、地形改造、召唤使役。 |

## 卡牌类型

| 术语 | 定义 |
|------|------|
| **【卡牌】** | 玩家打出以施展效果的基本单位，分为多种类型。 |
| **【魔法卡】** | 施法核心卡，卡组主体。打出即施放对应魔法，有基础效果，无需额外组合即可独立使用。卡组中存放原始版本，可被【语言卡】在本回合临时变为变体版本。 |
| **【语言卡】** | 本回合变体触发卡。打出后，本回合内后续打出的【魔法卡】变为对应语言的变体版本，回合结束后失效。 |
| **【姿势卡】** | 可选词条附加卡（词条卡之一）。搭配【魔法卡】使用，给本次施法追加释放形态相关的词条效果。 |
| **【材料卡】** | 可选词条附加卡（词条卡之一）。搭配【魔法卡】使用，给本次施法追加元素属性相关的词条效果。 |
| **【词条卡】** | 【姿势卡】与【材料卡】的统称，是可选的施法增强手段，可叠加使用。 |
| **【改良卡】** | 对本次施法进行增幅或改变效果，有一定失败概率，失败时魔法可能走形或产生反效果；消耗后不回收。 |
| **【建筑卡】** | 含「建筑」tag 的卡牌，打出后在指定【格子】上生成对应【建筑】。 |
| **【地形卡】** | 打出后改变战场地貌，影响敌我双方行动。 |
| **【行为牌】** | 敌方打出的卡牌，代表敌人的每一个具体行动。 |

## 施法系统

| 术语 | 定义 |
|------|------|
| **【法式】** | 【魔法卡】搭配【语言卡】、【词条卡】等打出后产生的完整施法组合效果。朴素施法（仅打出【魔法卡】）也构成最简单的法式。 |
| **【魔法变体】** | 【魔法卡】在【语言卡】的作用下本回合呈现的特殊形态，具有不同于原始版本的属性或特性加成。 |
| **【弃牌机会】** | 每回合玩家可用的换牌次数。每次使用可弃置任意数量手牌并补抽等量新牌，用完后本回合无法再换牌。 |
| **【法式图鉴】** | 记录玩家已发现的所有【法式】组合效果，供战前参考。 |
| **【吟唱法】** | 世界观中的魔法体系，以【魔法卡】为核心，通过语言变体与词条增强来丰富施法效果。 |
| **【魔法吟唱】** | 持续打出卡牌施法的行为。 |

## 卡牌词条

| 术语 | 效果说明 |
|------|---------|
| **【保留】** | 【回合】结束时不进入【弃牌堆】，保留在【手牌】中。 |
| **【消灭】** | 使用后直接从游戏中移除，不进入【弃牌堆】。 |
| **【开战】** | 进入【手牌】时立即触发效果。 |
| **【回收】** | 使用后不进入【弃牌堆】，直接回到【抽牌堆】顶部。 |
| **【终极】** | 使用后立即结束本【回合】【吟唱阶段】。 |
| **【蒸发】** | 使用后从游戏中移除（同【消灭】，待统一）。 |
| **【储存】** | 可跨回合积累效果的词条。（待补充） |

## 牌库与手牌

| 术语 | 定义 |
|------|------|
| **【牌库】** | 玩家全套卡牌的集合，战前可进行有限次数的调整。 |
| **【卡组】** | 同【牌库】，强调构筑视角。 |
| **【牌组】** | 同【牌库】，强调出牌视角。 |
| **【手牌】** | 每【回合】从【牌库】抽取的、当前可打出的卡牌。 |
| **【抽牌堆】** | 尚未抽取的剩余牌库。 |
| **【弃牌堆】** | 已打出或已弃置的卡牌暂存区，特定条件下可重新洗入【抽牌堆】。 |

## 费用系统

| 术语 | 定义 |
|------|------|
| **【魔力值】** | 双方（玩家与敌方）每【回合】可用于打出卡牌的资源，没有固定上限。每【回合】开始重置为当前的【自然回复量】，不叠加上一【回合】剩余。 |
| **【自然回复量】** | 每【回合】开始自动获得的基础【魔力值】，是可变属性，可被状态、buff、【建筑】效果等影响。 |
| **【魔力回复】** | 卡牌 tag。打出带有该 tag 的卡牌可在本【回合】立即获得额外【魔力值】，可来自卡组构造或场上【单位】产出。 |
| **【魔力】** | 【魔力值】的简称，也指单张卡牌的魔力消耗值。 |
| **【混乱度】** | 待重新设计（留坑）。原设计为关卡紧张程度指标，现考虑改为敌方单位规模体现或却废弃。 |

## 回合流程

| 术语 | 定义 |
|------|------|
| **【回合】** | 游戏基本时间单位，双方行动在一个回合内交替进行。 |
| **【抽牌阶段】** | 回合开始时从【牌库】抽取固定数量【手牌】的阶段。 |
| **【弃牌换卡阶段】** | 【吟唱阶段】前，玩家可使用【弃牌机会】弃置手牌并补抽新牌的阶段。 |
| **【吟唱阶段】** | 玩家消耗【魔力值】打出【手牌】、施放魔法的阶段。 |
| **【结算阶段】** | 双方行为同步结算、更新战场状态、地形天气效果生效的阶段。 |

## 建筑系统

| 术语 | 定义 |
|------|------|
| **【建筑】** | 【城之魔法使】的核心机制。通过【建筑卡】在指定【格子】上生成的永久性设施，每回合持续触发效果。 |
| **【箭塔】** | 建筑示例。每【回合】开始时，将 1 张【射击】加入【手牌】。 |
| **【射击】** | 由【箭塔】产出的卡牌，具体效果待定。 |
| **【建筑合成】** | 同一格子叠加建筑时可能触发的合并升级机制（待定）。 |
| **【建筑升级】** | 建筑叠加或条件满足后触发的增强机制（待定）。 |
| **【塔楼】** | 城防类建筑的统称，损毁会削弱【魔力值】产出。 |
| **【城墙】** | 需要保护的核心防线，特定段落可附加强化效果。 |
| **【城门】** | 城防结构组成部分（待补充）。 |

## 格子与地形

| 术语 | 定义 |
|------|------|
| **【格子】** | 战场的基本空间单位，用于部署【建筑】或执行行动。 |
| **【六边形格子】** | 战场格子的具体形状，采用六边形网格布局。 |
| **【归属属性】** | 格子的所属标记，决定哪方可在该格子上部署【建筑】或执行行动。 |
| **【己方区域】** | 归属己方的格子范围，初始建筑周围的格子默认归属己方。 |
| **【敌方区域】** | 归属敌方的格子范围。 |
| **【中立区域】** | 尚未归属任何一方的格子范围。 |
| **【护城区域】** | 城门外围的特殊防御区域（待补充）。 |

## 单位类型

| 术语 | 定义 |
|------|------|
| **【生物单位】** | 具有生命值、可被攻击的实体单位。 |
| **【建筑单位】** | 部署在格子上的建筑类实体，通常为永久存在。 |
| **【守卫造物】** | 召唤型友方单位，在【城墙】段落驻守并自动攻击。 |
| **【侦察精灵】** | 召唤型友方单位，可提前揭示敌方下一轮【行为牌】。 |
| **【侦察造物】** | 同【侦察精灵】（待统一名称）。 |
| **【魔法炮台】** | 召唤型友方单位，固定位置持续输出，消耗维持【魔力值】。 |
| **【护盾造物】** | 召唤型友方单位，为【城墙】附加护盾层。 |
| **【魔法使役】** | 召唤类造物的统称。 |
| **【造物】** | 同【魔法使役】。 |

## 关卡类型

| 术语 | 定义 |
|------|------|
| **【城防战】** | 守护特定城池，抵御若干波敌军攻势的关卡类型。 |
| **【野战支援】** | 玩家作为后方【吟唱使】，通过魔法影响正面战场走向的关卡类型。 |
| **【突围战】** | 保护友军从包围中撤退、控制撤退路线的关卡类型。 |
| **【决战关卡】** | 面对敌方【吟唱使】等特殊对手的关卡类型，对手同样会打出【法式】牌。 |

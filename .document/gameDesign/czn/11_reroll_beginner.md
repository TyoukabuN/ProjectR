## Reroll 目标强度分析

| 层级 | 角色 | 强度理由 |
|---|---|---|
| **SS** | Mei Lin | Rising Dragon Spire 无限叠层，单回合多段攻击，自摸牌，独立运作 |
| **SS** | Veronica | 全游戏最强摸牌引擎，0AP摸2张，为任何队伍提升效率 |
| **S** | Chizuru | Shadow of the Moon 无限虚空伤害叠层，build路径多样 |
| **S** | Hugo | Commence the Hunt 叠层追打，需攻击型队友配合 |
| **S** | Luke | 免费子弹生成，Dance of the Demon 无限叠层 |
| **S** | Khalipe | 护盾+AoE伤害兼备，Vulture Ejections，build灵活 |
| **A** | Haru | Anchor Shot 高爆发，与AP生成型队友配合出色 |
| **A** | Yuki | Inspiration 持续AoE，AP效率高，需摸牌支持 |
| **A** | Renoa | Dirge Bullets 追打，需弃牌辅助，多路径 |
| **A** | Rin | 单人DPS专家，维持 Dark Mist Stance，多段攻击，上手易精通难 |
| **A** | Kayron | Futility牌体系，主/副C均可，build和上手难度高 |
| **B** | Magna | 进攻型肉盾，护盾+反击+Vulnerable，牌费较高 |
| **B** | Orlea | 万能辅助（Buff/Debuff/治疗/护盾/伤害），适合长战，难度高 |

**设计启示**：
- 最顶级的角色都拥有"**自我成长飞轮**"（自摸牌/叠层无限化），无需依赖队友辅助即可运作
- SS级角色的共同特征：**摸牌效率** × **叠层上限无限** = 顶级强度
- "容易build但难精通"（Rin）vs "难build但上限高"（Kayron）——两种设计哲学都有空间

## 新手引导流程设计参考

> CZN 的引导设计采用"强制进入，分步解锁"模式，以下为完整流程：

### 快速引导（15~25分钟）

1. 下载游戏，登录账号（可绑定账号，无需访客）
2. 开场动画+序章第一战（快进按钮加速对话）
3. 序章第二战
4. 剧情关卡 → 抽卡教学 → **保底获得 Tressa（免费4星）**
5. Chaos Manifestation 教学（**不可跳过/快进/自动**，强制完整观看）
6. 进入主菜单 → 领取邮箱奖励
7. 前往 Rescue → **Special Request Rescue Banner**（新手专属8锚=10抽）
8. 完成前3场战斗任务，领取奖励 → **继续抽 Amplified Distress Signal 限定Banner**
9. 如不满意：设置 → 重置服务器（数据清零，无需新账号）
10. 重复直到满意

### 深度引导（40~60分钟，更多资源）

- 在快速流程基础上：推进至 **Captain Level 10**
- 完成 Growth Guide 第一页所有任务
- 解锁 Events 页面 → 额外资源 → 限定Banner可多10抽
- 总计：可完成 Special Request Rescue（50抽保底5星）+ 限定Banner 20抽

### 引导设计特点（对比参考）
| 特征 | CZN做法 | 设计意义 |
|---|---|---|
| 强制教学 | Chaos Manifestation 不可跳过 | 确保玩家理解核心 Roguelite 机制 |
| 保底礼包 | 新手50抽内必出5星 | 降低流失率，保证体验上限 |
| 免费角色 | Tressa（4星）教学赠送 | 让玩家有完整体验而无需氪金 |
| 渐进解锁 | Level 10解锁Events | 分批给予成就感，延长留存 |
| 重置机制 | 账号内一键Reset | 降低Reroll门槛，留住hardcore玩家 |

## Reroll 货币与机制概要

| 货币 | 用途 | 获取方式 |
|---|---|---|
| **Crystal（水晶）** | 普通抽卡货币 | 日常/任务/活动 |
| **Rescue Anchor（救援锚）** | 高级抽卡货币（10锚=10抽常规；新手8锚=10抽） | 邮件奖励/任务/充值 |
| **Policy Points** | 购买政策报告/礼物 | 批准政策获得 |
| **Zeronium** | 特殊商店货币 | 活动/特定内容 |

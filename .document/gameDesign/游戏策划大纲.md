# 游戏策划大纲

---

## 一、游戏定位与核心印象

### 核心角色：魔法吟唱使
魔法吟唱使是魔法使的顶点，深谙对现实具象化影响之道。扭动身体，挥舞四肢，改变手势，无间断地进行魔法吟唱。可以是连绵不断的攻势，可以是不绝于耳的祝福，可以是鳞次栉比的防御。在各种战斗场合他们都能以一己之力改变战局。

### 核心场景印象

**场景一：城防魔法吟唱**
玩家是一个魔法吟唱使，站在城墙塔楼上持续吟唱魔法，抵御靠近城墙并试图攻城的敌人。
- 玩法围绕城池展开
- 可以改变城墙的各种属性来防御攻击
- 召唤城防武器来反击
- 改变地形，对敌方赋予移动方面的碍害 Debuff
- 改变天气，对敌方赋予防御方面的碍害 Debuff

**场景二：战术魔法吟唱**
玩家在一场大规模战斗中，在后方不断吟唱魔法来决定战斗的走向。
- 使用地形改变魔法，减缓敌人前进的脚步
- 使用广域攻击魔法，高效消灭敌人
- 使用广域 Buff 技能强化友方
- 召唤魔法使役与造物，辅佐防御

## 二、参考游戏分析

| 游戏 | 参考要素 |
|------|----------|
| **炉石传说 / 影之诗** | 手牌机制：从牌库抽卡、费用管理、手牌组合对决的核心框架；费用逐回合递增的节奏设计 |
| **Monster Train** | 波次防御结构：敌人分波冲击、玩家用牌应对，防守节点明确 |
| **杀戮尖塔** | Roguelike 卡组构筑：战斗中逐步完善牌组、单局决策深度 |
| **Magicka** | 元素组合施法：不同元素（语言/姿势/材料）叠加组合产生迥异的魔法效果 |
| **Faeria** | 地形卡牌联动：通过改变战场地貌影响策略走向 |

> 本游戏是对上述设计的创新融合，最核心的独特点是：**敌人的每一个行为本身就是打出一张卡牌**，关卡与玩家形成真正意义上的"对局"。

## 三、核心玩法设计

### 3.1 卡牌施法系统

魔法吟唱使通过卡牌的组合来完成法术的吟唱与施展。卡牌分为以下类型：

| 卡牌类型 | 说明 |
|----------|------|
| **语言卡** | 描述施法所用的魔法语种，决定魔法的属性流派与基础规则（如精灵语、魔神语、龙语） |
| **姿势卡** | 描述施法者的身体动作，决定魔法的释放形态（如扩散、集中、持续、爆发） |
| **材料卡** | 描述魔法的物质构成（如火焰、寒冰、石土、风暴、圣光、暗影） |
| **改良卡** | 临时插入当前法式中，增幅或改变魔法效果，存在一定失败风险 |

#### 施法公式
魔法由固定组合公式触发，例如：
- `语言：精灵语` + `姿势：挥臂` + `材料：烈焰` = 精灵火焰弹（精准单体、灼烧持续）
- `语言：龙语` + `姿势：绕转` + `材料：石土` = 龙脊隆起（地形更宽广、持续时间更长）
- `语言：魔神语` + `姿势：压掌` + `材料：寒冰` = 冰封诅咒（附魔效果更强、但对友方也有冰冻风险）

公式可在战前整理，战斗中从牌库抽卡，凑出需要的组合以应对战局。

#### 临时改良
在标准法式中临时插入改良卡，可增强效果（如范围扩大、伤害翻倍），但：
- 改良卡有一定失败概率，失败时魔法可能走形或产生反效果
- 改良卡消耗后不回收，具有一次性风险成本

### 3.2 关卡即对手——双方对牌机制

关卡中的敌人不仅仅是数值上的障碍，其**每一个行为都对应一张行为牌**，公开亮出后在下一回合执行。

| 敌方牌示例 | 效果 |
|------------|------|
| **冲锋** | 某小队本回合移动速度+2，强行突破前线地形障碍 |
| **箭雨覆盖** | 对城墙某段造成持续伤害，压制城防反击 |
| **包围推进** | 多路同时施压，分散玩家吟唱焦点 |
| **攻城器进场** | 召唤投石车，对城墙本体造成结构伤害 |
| **溃退重整** | 敌方撤退，为下一波更强的推进蓄力（需玩家判断是否追击还是补强防御） |

玩家可以通过侦察、预判牌面，提前构造对应的法式应对。

**敌方行为牌的数量由"混乱度"决定**（详见 4.6 混乱度系统）。

### 3.3 城防结构

城防由多个可交互节点构成：

- **城墙本体**：有耐久度，可通过强化魔法提升，被攻城器打击时受损
- **塔楼**：玩家吟唱施法的立足点，塔楼受损会影响吟唱效率（手牌上限/抽牌速度）
- **城门**：核心保护目标，城门被突破即战败
- **护城区域**：城墙外的缓冲地带，可在此设置地形陷阱

## 四、系统设计

### 4.1 牌库构筑系统
- 玩家拥有一套基础牌库，战前可进行有限次数的组合调整
- 关卡结束后获得新卡牌或对现有卡牌进行升级
- 每场战斗中每回合从牌库抽取固定数量的牌，手牌打出后需等待下一轮补充

### 4.2 法式图鉴系统
- 玩家在探索中逐渐解锁更多卡牌组合公式
- 部分稀有公式需要特定三张牌同时在手才能触发隐藏魔法效果
- 图鉴记录玩家已发现的所有法式，供战前参考

### 4.3 地形改造系统
城防战中，玩家可通过地形卡对战场环境进行主动改造：

| 地形魔法 | 效果 |
|----------|------|
| **隆起** | 在指定位置生成石墙/土丘，阻挡敌方通路 |
| **沼泽化** | 将地面变为泥泞，敌方移动速度大幅下降 |
| **冰封地面** | 地面结冰，敌方移动时有一定概率跌倒（停顿一回合） |
| **烈焰地毯** | 在地表铺设持续燃烧区域，对经过的敌人持续造成伤害 |
| **深坑** | 在地面制造陷阱深坑，轻型单位可能坠入（永久清除） |

### 4.4 召唤系统
玩家可通过特定法式召唤以下类型的使役造物辅助城防：

- **守卫造物**：在城墙段落驻守，有自己的攻击行为
- **侦察精灵**：前出探查敌方牌面，提前揭示下一轮敌方行为牌
- **魔法炮台**：固定位置持续输出，消耗一定维持魔力
- **护盾造物**：为城墙特定段落附加护盾层，承受额外伤害

### 4.5 天气/环境系统
特定魔法或特定回合数后，战场天气将发生变化，影响双方战力：

| 天气 | 对敌效果 | 对友效果 |
|------|----------|----------|
| **暴风雪** | 敌方移速-30%，远程精度下降 | 火系魔法记忆量消耗+1 |
| **浓雾** | 敌方视野受限，无法提前亮牌 | 侦察造物效果增强 |
| **烈日** | 敌方无特殊影响 | 火系、光系魔法强化 |
| **雷暴** | 敌方重甲单位移速受阻 | 雷系魔法可连锁触发 |

天气可被玩家主动触发（消耗高额魔法记忆量），也可能作为关卡机制自然变化。

### 4.6 魔法记忆量——玩家施法费用系统

**魔法记忆量**是玩家每回合可用于施法的资源上限，体现吟唱使当前能同时记住并驾驭的法式复杂度。

#### 基本规则（类炉石传说法力水晶）
- 战斗开始时，魔法记忆量为 **1**
- 每经过一个回合，当前上限 **+1**，直至达到关卡设定的**最大上限**（通常为 10）
- 每回合开始时，记忆量**完全恢复**至当前上限
- 每张卡牌（语言/姿势/材料/改良）都有对应的**记忆消耗值**
- 当回合剩余记忆量不足以支付下一张牌时，玩家无法再出牌

#### 记忆量调节
- 部分卡牌效果可临时增加本回合记忆量（如"冥想"）
- 塔楼损毁会永久降低记忆量上限（施法阵地被破坏，专注力下滑）
- 特定法式（如"集中精神"）可在下一回合额外获得记忆量

#### 组合法式的记忆消耗
一套完整法式的总消耗 = 各张卡牌消耗之和，例如：
- `语言：精灵语（1）` + `姿势：挥臂（1）` + `材料：烈焰（2）` = **共消耗 4**
- `语言：龙语（2）` + `姿势：绕转（2）` + `材料：石土（3）` = **共消耗 7**（需要足够高的回合数才能施展）

这意味着**强力的法式只能在战斗中后期才能发动**，与战场形势的演变形成自然的节奏呼应。

### 4.7 混乱度——关卡施压费用系统

**混乱度**是关卡/战场的紧张程度指标，决定敌方每回合能够打出的行为牌数量。混乱度越高，战场局面越失控，敌方行动越频繁。

#### 基本规则
- 战斗开始时，混乱度有一个**初始值**（由关卡难度决定），并设有**当前上限**
- 每回合结算后，混乱度 **+1**（战场态势自然恶化）
- 混乱度当前上限随回合数缓慢提升（敌方援军陆续到位，局面越来越难以控制）
- 当前混乱度 ÷ 混乱度上限 = 敌方本回合可打出的**行为牌数量比例**

#### 玩家如何压制混乱度
玩家**成功化解**敌方行为牌，可以阻止混乱度增长，甚至降低混乱度：

| 应对结果 | 混乱度变化 |
|----------|-----------|
| 完全化解一张敌方牌 | 混乱度 **-1**（局面稳定） |
| 部分化解（减轻伤害） | 混乱度**不变** |
| 未应对，敌方牌生效 | 混乱度 **+2**（局面恶化加剧） |
| 打出特定"镇压"法式 | 混乱度 **-3**（主动平乱，消耗极高记忆量） |

#### 混乱度对敌方出牌的影响
| 混乱度区间 | 敌方每回合行为牌数量 | 备注 |
|------------|---------------------|------|
| 1 ~ 3 | 1 张 | 前期，战局尚可控 |
| 4 ~ 6 | 2 张 | 中期，开始需要取舍 |
| 7 ~ 9 | 3 张 | 后期，压力倍增 |
| 10（上限） | 4 张 + 特殊事件触发 | 崩溃边缘，危机状态 |

#### 核心张力
> 玩家面临的根本选择是：**用有限的魔法记忆量去化解敌方牌（压制混乱度），还是发动强力攻势（消耗记忆量输出）**。
> 专心防御则输出不足，任由混乱度攀升则战局很快失控——两者之间的动态平衡即是本游戏的核心博弈。

## 五、战斗节奏设计

### 5.1 回合结构（单回合流程）

1. **回合开始**：魔法记忆量恢复至当前上限，混乱度 +1
2. **抽牌阶段**：从牌库抽取固定数量手牌
3. **敌方亮牌阶段**：关卡根据当前混乱度决定本回合打出的行为牌数量，并公开亮出（玩家可提前预判）
4. **吟唱阶段**：玩家消耗魔法记忆量打出手牌，组合法式，施展魔法
5. **结算阶段**：双方行为同步结算，根据化解结果更新混乱度，地形/天气效果持续生效
6. **回合终了**：用过的手牌进入弃牌堆，下一回合重新抽牌

### 5.2 压力节奏——波次设计
- 前期波次：敌方行为牌数量少，组合简单，给玩家熟悉牌式的时间
- 中期波次：敌方开始打出组合牌（冲锋+箭雨同时进行），玩家需多线应对
- 后期/Boss波次：敌方亮出大招牌（攻城器全线压进+溃退重整诱骗），决策压力拉满

## 六、关卡与叙事框架

### 6.1 关卡类型
- **城防战**：守护特定城池，抵御若干波敌军攻势
- **野战支援**：玩家作为后方吟唱使，通过魔法影响正面战场走向
- **突围战**：保护友军从包围中撤退，控制撤退路线
- **决战关卡**：面对敌方吟唱使等特殊对手，对手同样会打出法式牌

### 6.2 叙事背景（待细化）
- 世界中存在一种古老的魔法体系——**吟唱法**，由魔法语种（精灵语/龙语/魔神语等）、身体动作与自然材料三要素构成
- 魔法吟唱使是习得这套体系巅峰的极少数人
- 玩家所在的势力面临外敌入侵，玩家以吟唱使身份参与一系列关键防守战役

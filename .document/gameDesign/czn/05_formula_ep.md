
> 来源：https://www.gamekee.com/czn/682515.html

### 基于攻击的伤害
```
实际伤害 = 卡牌倍率 × 实际攻击力 × 0.35
         × (1 + 属性伤害) × (1 + 持续伤害) × (1 + 额外伤害)
         × 敌人防御倍率 × 暴击伤害
```

### 基于防御的伤害
```
实际伤害 = 卡牌倍率 × (实际攻击力×0.3 + 实际防御力×2.1) × 0.35
         × (1 + 属性伤害) × (1 + 持续伤害) × (1 + 额外伤害)
         × 敌人防御倍率 × 暴击伤害
```

### 护盾/治疗
```
实际护盾/治疗量 = 卡牌倍率 × (基础防御力 + 实际防御力) × 0.5 × 0.3
```

### 关键倍率规则
- **增加XX%** = 乘算（通用乘法倍率）；**+XX%** = 加算（通用加法倍率）
- **弱点攻击**：基础 ×1.25；**脆弱**：×1.5；**削弱（虚弱）**：×0.75
- 额外攻击、反击、持续伤害均吃属性伤害加成；额外攻击另外吃暴击
- 所有"回合外伤害"均不享受弱点加成

### 对本游戏启示
- 防御型角色的"防御出伤"公式中防御权重更高（×2.1），值得借鉴设计"防守反击"类法式
- 乘算与加算并存的公式决定了卡组多样性：纯堆乘算/纯堆加算各有适合的法式类型

## 十三、自我意识点数（EP）系统

> 来源：https://www.gamekee.com/czn/681736.html

### 基础规则
- EP = 独立资源，用于觉醒技能和伙伴主动技能
- 战斗开始初始 **2.5 格**；每回合开始自然回复 **0.5 格**
- EP不重置，通关整个卡厄思才清零
- 使用技能时**不减少敌方行动计数**

### EP回复规则（经实测）
| 触发条件 | EP回复量 |
|----------|----------|
| 0费牌 | 不回复 |
| 1费牌 | 约10% |
| 2费牌 | 约25% |
| 3费牌 | 约40% |
| 击败怪物 | 约25% |
| 击破韧性 | 不回复 |
| 装备自动发动的牌 | 不回复 |
| 被改变费用的牌 | 按改变后费用计算 |

### EP回复的关键结论
- 一回合打完3点AP + 击败一只怪物 ≈ 稳定回复1点EP
- **高费牌是EP回复的核心**：打3费牌比三张1费牌回复更多

### 对本游戏启示
- 本游戏的「觉醒/大招」类技能可参考此设计：高费法式打出时顺带充能大招资源
- "高费奖励"机制不仅仅是费用返还，还可以是"充能特殊技能槽"

# Dominion 基础版王国卡图鉴

> 以下为基础版（第二版）全部 25 张王国卡的效果与策略说明。

---

## 3.1 $2 费用卡

### Cellar（地窖）
| 属性 | 内容 |
|------|------|
| **费用** | $2 |
| **类型** | Action |
| **效果** | +1 Action；弃置任意数量的手牌，然后抽取等量新牌 |

**策略分析**：
- 非终端卡（Cantrip 类），不消耗行动点净值
- 弃置-重抽机制允许手牌循环，滤掉当前不需要的牌
- 在有弃置奖励的王国中特别有价值（弃到好效果触发）
- 可以配合需要弃牌的卡牌形成组合

---

### Chapel（礼拜堂）
| 属性 | 内容 |
|------|------|
| **费用** | $2 |
| **类型** | Action |
| **效果** | Trash（移出游戏）手牌中最多 4 张卡 |

**策略分析**：
- **Dominion 最强的清牌卡之一**，甚至是全游戏最强卡之一
- 最优策略：开局尽早买到 Chapel，立即开始清理初始的 Copper 和 Estate
- 目标是将牌库缩减到 5-7 张精锐卡，实现每回合稳定抽到强力卡
- 是 Engine 策略的重要基石
- Chapel 本身被清理掉后，牌库就已经干净了（一次性工具）

---

### Moat（护城河）
| 属性 | 内容 |
|------|------|
| **费用** | $2 |
| **类型** | Action - Reaction |
| **效果** | +2 Cards；（当他人打出 Attack 时，可展示此卡，免疫该攻击）|

**策略分析**：
- Reaction 效果**无需消耗行动点**，从手牌直接激活
- 在有强力 Attack 卡（如 Witch）的王国中，Moat 是唯一的防御手段
- 打出时 +2 Cards 的效果也很实用，是较好的终端抽牌卡
- 缺点：既是 Attack 防御，又是抽牌卡，定位稍混乱

## 3.2 $3 费用卡

### Harbinger（先驱）
| 属性 | 内容 |
|------|------|
| **费用** | $3 |
| **类型** | Action |
| **效果** | +1 Card；+1 Action；可以将弃牌堆中的一张牌放到牌库顶 |

**策略分析**：
- Cantrip 类，不消耗行动点
- 将弃牌堆中的好牌放到牌库顶，实现有效的"牌库挖掘"
- 可以提前拿到关键卡，加速引擎运转

---

### Merchant（商人）
| 属性 | 内容 |
|------|------|
| **费用** | $3 |
| **类型** | Action |
| **效果** | +1 Card；+1 Action；本回合第一次打出 Silver 时 +$1 |

**策略分析**：
- Cantrip 类
- 与 Silver 的协同使每张 Silver 的价值从 $2 提升到 $3
- 多张 Merchant 叠加时，只有第一张享有 Silver 加成（设计很精妙）

---

### Vassal（封臣）
| 属性 | 内容 |
|------|------|
| **费用** | $3 |
| **类型** | Action |
| **效果** | +$2；弃置牌库顶，若为 Action 卡则可选择直接打出（不消耗行动点）|

**策略分析**：
- 提供 $2 的同时，有概率免费打出一张 Action 卡
- 与高价值 Action 卡组合时效果惊人（如 Vassal + Vassal 链）
- 运气成分较强，但期望值很高

---

### Village（村庄）
| 属性 | 内容 |
|------|------|
| **费用** | $3 |
| **类型** | Action |
| **效果** | +1 Card；+2 Actions |

**策略分析**：
- **"Village"类卡的原型**，是 Engine 策略的基础构件
- +2 Actions 是行动点净增加，可以支撑多张终端卡
- 经典搭配：Village（+2行动）+ Smithy（终端抽3牌）
- 本身只抽1张牌，不提供金钱，需要配合其他卡使用

---

### Workshop（工坊）
| 属性 | 内容 |
|------|------|
| **费用** | $3 |
| **类型** | Action |
| **效果** | 获得一张费用 $4 以下的卡 |

**策略分析**：
- 获得卡牌，而非购买（不消耗 Buy 机会）
- 可以稳定获得 Silver 或 $4 的 Action 卡
- 配合 Gardens（花园）构成"冲刺"策略：Workshop 快速堆大牌库，Gardens 因总卡数多而得高分

---

## 3.3 $4 费用卡

### Bureaucrat（官僚）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action - Attack |
| **效果** | 将 Silver 放到牌库顶；其他玩家展示手牌中的 Victory 卡并放到各自牌库顶（无则展示整手牌）|

**策略分析**：
- 给自己加速 Silver，同时干扰对手（强迫对手下一回合抽到死牌）
- 对正在"变绿"的对手伤害尤其大

---

### Gardens（花园）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Victory |
| **效果** | 游戏结束时：牌库每 10 张卡 = 1VP |

**策略分析**：
- **Alt-VP 卡的经典案例**，颠覆了"牌库要精简"的通常逻辑
- Gardens 策略鼓励反向操作：**故意让牌库膨胀**来获得高分
- 通常配合 Workshop 等获得卡的能力快速堆牌
- 配合冲刺策略（Rush）可以快速结束游戏并占据大量 Gardens

---

### Militia（民兵）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action - Attack |
| **效果** | +$2；其他每位玩家弃牌至手牌只剩 3 张 |

**策略分析**：
- 提供 $2 的同时进行手牌破坏攻击
- 对 Engine 玩家伤害尤大（手牌大小是引擎的命脉）
- 对大钱玩家伤害有限（抽少几张牌影响不大）

---

### Moneylender（放债人）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action |
| **效果** | 可以 Trash 一张 Copper；若如此，+$3 |

**策略分析**：
- 清理初始 Copper 的同时提供丰厚的 $3
- 是一种"主动清牌 + 临时资源"的设计，每次使用都是权衡

---

### Poacher（盗猎者）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action |
| **效果** | +1 Card；+1 Action；+$1；每有一个空的 Supply 堆，弃置一张手牌 |

**策略分析**：
- Cantrip 附带 +$1，非常高效
- 随着游戏进行，空堆增多，弃牌惩罚也增大——后期效果下降
- 在早中期是强力的非终端金钱生产者

---

### Remodel（改造）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action |
| **效果** | Trash 手牌一张，获得费用最多比其高 $2 的卡 |

**策略分析**：
- "以旧换新"机制：将低价值卡升级为更强的卡
- 可以将 Estate 改造为 Silver，将 Copper 改造为 Estate（游戏结束前的计分卡）
- 灵活性高，是构建过渡策略的重要工具

---

### Smithy（铁匠）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action |
| **效果** | +3 Cards |

**策略分析**：
- **最经典的终端抽牌卡**
- 一次抽 3 张，大幅增加手牌规模
- 大钱策略中最常见的辅助卡："大钱+Smithy"是最有效的纯大钱优化方案
- 需要行动点支持（Village 等），否则每回合只能打一张

---

### Throne Room（王座间）
| 属性 | 内容 |
|------|------|
| **费用** | $4 |
| **类型** | Action |
| **效果** | 可以打出一张手牌中的 Action 卡两次 |

**策略分析**：
- **"倍增卡"原型**，是 Combo 策略的核心工具
- Throne Room + Smithy = 抽 6 张牌
- Throne Room + Throne Room = 一张卡被打出 4 次（链式叠加）
- 在有强力终端卡的王国中威力巨大

---

## 3.4 $5 费用卡

### Bandit（强盗）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action - Attack |
| **效果** | 获得 Gold；其他玩家展示牌库顶 2 张，Trash 其中一张 Treasure（非 Copper），弃置其余 |

**策略分析**：
- 给自己稳定获得 Gold，同时破坏对手的高价值 Treasure
- 对大钱玩家非常有效（摧毁 Gold/Silver）

---

### Council Room（议事厅）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | +4 Cards；+1 Buy；其他每位玩家抽 1 张牌 |

**策略分析**：
- 强力终端抽牌卡，+4 张牌非常可观
- 缺点：给对手也抽了牌，在多人游戏中可能会帮助对手
- 额外的 +1 Buy 在有多个购买目标时很有价值

---

### Festival（节日）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | +2 Actions；+1 Buy；+$2 |

**策略分析**：
- Village 的升级版：同样提供 +2 Actions，还额外附带 +$2 和 +1 Buy
- 是资源密集型 Engine 的核心，解决了行动点和金钱同时不足的问题
- 高费用但价值明显，很值

---

### Laboratory（实验室）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | +2 Cards；+1 Action |

**策略分析**：
- **最强的非终端抽牌卡之一**
- 自带 +1 Action，不依赖 Village 也能独立链式打出
- 叠加多张 Laboratory 可以在一回合内抽出大量手牌
- 是 Engine 策略的顶尖构件

---

### Library（图书馆）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | 抽牌直到手牌共 7 张；遇到 Action 卡时可选择跳过（放入临时区，回合结束后丢弃）|

**策略分析**：
- 特殊的"抽到X张"机制，保证手牌至少有 7 张非 Action 牌
- 可以跳过不需要的 Action 卡（过滤机制）
- 在有 Treasure-heavy 王国中表现极佳

---

### Market（市场）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | +1 Card；+1 Action；+1 Buy；+$1 |

**策略分析**：
- **全能型 Cantrip**，四个效果均衡
- +1 Buy 是最稀缺的资源之一，Market 是基础版中少数提供它的卡
- 多张 Market 叠加时，Buy 和金钱都会显著增加

---

### Mine（矿山）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | Trash 手牌中一张 Treasure，获得费用最多高 $3 的 Treasure 放入手牌 |

**策略分析**：
- Treasure 版的 Remodel，专门升级财宝卡
- Copper → Silver，Silver → Gold 是最典型的路径
- 获得的 Treasure 直接进入手牌（本回合可立即使用）

---

### Sentry（哨兵）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action |
| **效果** | +1 Card；+1 Action；查看牌库顶 2 张；可以选择 Trash、弃置或放回（任意顺序）|

**策略分析**：
- Cantrip + 牌库顶操控 + 清牌能力
- 兼具抽牌、过滤和清牌三种功能，非常灵活
- 可以把 Curse、Estate 等垃圾牌清掉，同时保留好牌

---

### Witch（女巫）
| 属性 | 内容 |
|------|------|
| **费用** | $5 |
| **类型** | Action - Attack |
| **效果** | +2 Cards；其他每位玩家各获得一张 Curse |

**策略分析**：
- **基础版最强的攻击卡，也是全游戏最具代表性的 Attack 卡之一**
- Curse 是 Dominion 中最强的攻击形式：稀释对手牌库 + 扣分
- 自带 +2 Cards，本身也是强力抽牌卡
- "有 Witch 的王国"会从根本上改变策略重心：Curse 的争夺和防御成为核心

---

## 3.5 $6 费用卡

### Artisan（工匠）
| 属性 | 内容 |
|------|------|
| **费用** | $6 |
| **类型** | Action |
| **效果** | 获得费用 $5 以下的卡进入手牌；将手牌中一张卡放到牌库顶 |

**策略分析**：
- 强力获得卡，获得任意 $5 以下的卡直接进手牌（本回合即可使用）
- 放牌到牌库顶确保下回合抽到关键卡
- 可以在同回合内自搜：Artisan 获得 Province，再把 Province 放牌库顶（下回合直接抽到）
